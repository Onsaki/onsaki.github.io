计划把主题的图库更新一波4K超高清然后再压到1K

发表于2025-08-22本文字数771字阅读时长1 分钟

首页关于hexo

# 起因

主题默认的图库画质有点太糊了，原图虽然是1080P但画质极低，小屏幕上还好，但大屏幕上看就糊的不像个提速降费之后的体验。所以打算把主题的图源全部替换为4K高清图。

# 图源

现在是打算从收藏的wlop的画里面选出来一部分做图源，但是wlop的图总感觉和主题的风格不太匹配，大多数图抖太灰暗了，不符合我阳光开朗大男孩的人设。

所以还得找点好看的图，暂时没头绪，反正也不急，我这blog也没人看，慢慢收集就是了，争取找他个百来张图。

# 图床

继续用github做图床，目前来看我自己的联通家宽裸连github问题不大，加载速度是稍微慢了一点，但反正也没人看，等真的有人看了再研究研究shokax。

# 发牢骚

shokax我是一直想换过去，但是那个感觉对小白过于不友好，其实对我这种什么都不懂的小白，直接git安装可能是更好的办法。shokax那个即食罐头我反复配置了三四十次了，没有一次能用的，不是少点这就是少点那，把我心态搞崩了，不然我也不会一直在这个停更几年的主题上靠着豆包修修改改。

但凡shokax有一次配置成功我都早就把主题换成shokax了，希望那个文档能稍微照顾一下纯小白吧。

但不照顾也无所谓，本来就是为爱发电的项目，而且shoka我现在还能凑合着用，什么时候彻底不能用了再豁出命研究shokax

# 尾巴

最初建站的目的就是说一些在国内平台不方便说的话，主要就是骂傻逼。

但建站之后又懒得骂了。

站点的名字是Onsaki，其实就是蒙语的“你放屁”的音，我只是把q改成了k，让脏话不那么明显。

至于为什么懒得骂了，是因为这几年的各种对立我才想明白。脑残、脑瘫是疾病，是可以积极治疗减轻症状的。

但傻逼就是傻逼，傻逼不是病，傻逼就是傻逼。
