# Dreamroom Productions, Inc.

缩写DRP，日文是ドリームルーム プロダクション社，中文是梦想之屋制作公司。2004年正式成立。公司地址:美国加利福利亚州洛杉矶市圣.菲格罗亚街445号2600室，邮编CA90017。业务范围为企画、制作和编辑无修正的成人视频，通过网络渠道，为付费会员提供下载服务。其除了自身会独立企画、制作制作视频外，还负责接受其他的公司的业务委托，负责她们的作品在互联网的发行和版权管理，这也就是为什么大家看到很多光盘作品的片头和片中都会出现DRP标志的原因。而且这种合作是双向的，其他的公司把售卖的光盘和视频委托给DRP和旗下的子公司在网络发行；反之，DRP和旗下的子公司，也会把自己制作的视频给有合作关系的其他公司去制作光盘上架售卖。DRP目前是日本海外无修正的成人视频最大的网络配信集团。集团目前下辖很多子公司，DRP和这些旗下子公司，给客户主要提供avi，wmv，mp4这3种视频格式的视频下载。

# カリビアンコム

英文是Caribbeancom，中文是加勒比。注册地址是美国加州。是DRP旗下的最大的一家子公司，号称“无修正アダルトサイト?エロ动画 NO.1”，就是无修正成人视频网络配信商第一。Caribbeancom，在2001年成立，2004年成为DRP的子公司，依托DRP强大的后援，加上自己独立企画的作品，迅速成为现在最大的网络配信商。到2013年已经出品3000多部作品，保持着一周六部作品的更新速度，早期主要提供720P分辨率的作品，现在已经全面转向提供1080P分辨率的。视频格式为WMV和MP4两种格式。旗下还有两家子公司，caribbeancomgirl和caribbeancompr。

# caribbeancomgirl

番号形式：一串数字，如090512-121（永沢まお 連續!潮吹大洪水 後編）。
