# Dreamroom Productions, Inc.

缩写 DRP，日文是ドリームルーム プロダクション社，中文是梦想之屋制作公司。2004 年正式成立。公司地址：美国加利福利亚州洛杉矶市圣。菲格罗亚街 445 号 2600 室，邮编 CA90017。业务范围为企画、制作和编辑无修正的成人视频，通过网络渠道，为付费会员提供下载服务。其除了自身会独立企画、制作制作视频外，还负责接受其他的公司的业务委托，负责她们的作品在互联网的发行和版权管理，这也就是为什么大家看到很多光盘作品的片头和片中都会出现 DRP 标志的原因。而且这种合作是双向的，其他的公司把售卖的光盘和视频委托给 DRP 和旗下的子公司在网络发行；反之，DRP 和旗下的子公司，也会把自己制作的视频给有合作关系的其他公司去制作光盘上架售卖。DRP 目前是日本海外无修正的成人视频最大的网络配信集团。集团目前下辖很多子公司，DRP 和这些旗下子公司，给客户主要提供 avi，wmv，mp4 这 3 种视频格式的视频下载。

# カリビアンコム

英文是 Caribbeancom，中文是加勒比。注册地址是美国加州。是 DRP 旗下的最大的一家子公司，号称 “无修正アダルトサイト？エロ动画 NO.1”，就是无修正成人视频网络配信商第一。Caribbeancom，在 2001 年成立，2004 年成为 DRP 的子公司，依托 DRP 强大的后援，加上自己独立企画的作品，迅速成为现在最大的网络配信商。到 2013 年已经出品 3000 多部作品，保持着一周六部作品的更新速度，早期主要提供 720P 分辨率的作品，现在已经全面转向提供 1080P 分辨率的。视频格式为 WMV 和 MP4 两种格式。旗下还有两家子公司，caribbeancomgirl 和 caribbeancompr。

# caribbeancomgirl

一个以提供在线视频表演为主营的子公司，能提供大量在线实时视频表演。每天都会推出几个专业的女优的在线视频表演，如果在里面看到有自己心仪的女优，是可以付费看她们的在线实时视频表演的，大受欢迎！
caribbeancompr, 是加勒比在 2004 年加入 DRP 以后成立的，片源全部来自 DRP, 全是完整没有切割过的，所以会员下载到的都是完整的，但是下载不实行付费包年制，而是下载一部需要单独付费 30 美元左右。至 2013 年已经出品了接近 1500 部了，也是一周六部的更新速度。国内很少接触。
番号形式：一串数字，如 090512-121（永沢まお 連續！潮吹大洪水 後編）。
