# 以前是以前

这几天用豆包的 AI 编程用的飞起，接连解决了 shoka 主题的超链接样式问题、修复 polyfill.js 失效的问题、给主题建了自己的图床并修改了图片加载的一些配置。

我以为我可以了，有豆包 AI 在，hexo 上没有我解决不了的问题了。

# 现在是现在

2025 年 8 月 5 日傍晚 19 点 45 分，我突然想把博客的评论系统从 valine 换成 waline，于是信心满满的打开豆包 AI，一次上传的主题的相关配置代码。

现在是 5 月 6 日凌晨 4 点整，豆包 AI 把我整不会了。

我把我能提交的代码全提交了，就差把整个主题文件全给他了，但按照他的步骤一步一步修改下来之后依然无法顺利的迁移到 waline。

我都不知道这几个小时我干了什么...

做了几个小时的无用功。

不知道 Github Copilot 能不能完美的解决我的问题。

主要是豆包支持上传的文件类型不多，而且有文件数量和大小限制。

不清楚 Github Copilot 付费版是不是可以无限量的分析代码，然后把整个主题分析一遍，帮我达成我的要求。

# 明天

看看哪里能租到 Github Copilot 号不，要是租不到就先用免费版的试试。
