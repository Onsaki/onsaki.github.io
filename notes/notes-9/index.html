只因为一个命令，被自己蠢到哭

发表于 2025-07-24
  • Onsaki
首页 闲散随笔

从去年博客复活之后又停更了大半年，真相只有一个：博客主题没弄明白。

现在用的这个主题的原作者几年前开始停更之后主题处在半瘫痪状态，很多东西作者忙于工作没时间修，我又没能力解决，所以博客在 hexo 复活之后就卡在了 99% 进度，最后的 1% 就是搜索功能。

因为博客只提供的 algolia 的搜索，而我又第一次弄这种需要在外站操作的情况，又碰巧 algolia 的网站全英文，而我是个丈育，英文全不懂，又碰巧主题原作者停更之后 algolia 的网站经历过一次改版，新建账户后没办法按主题文档介绍的那样一步一步操作，会直接进入一个五步的创建某个东西的流程。

但是！创建的东西和我需要用的功能完全无关，要选一堆我不知道干嘛的东西。114514 个小时候我终于按文档弄到了需要的 API 密钥，但是一直无法显示搜索结果。

于是博客进度卡在了 99.9%...

今天，我痛定思痛把主题从 shoka 换到了魔改版的 shokax，然后再去读主题的文档发现一个卡了我将近一年的蠢问题。

我在配置好 algolia 的 API 密钥之后，忘了执行 “hexo algolia” 这个命令，直接导致博客的数据没有上传到 algolia，所以之前虽然配置了 API 密钥，但一直无法在博客完成搜索。

我就因为少执行了一个命令，卡在 99.5% 的进度将近一年！！！

真的被自己蠢哭 QAQ~

以后我一定仔细阅读文档，再漏字少行我就是🐶！
